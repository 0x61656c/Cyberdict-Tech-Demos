Cybersecurity Section


Term Definition
Cybersecurity A set of techniques used to protect the integrity of networks, programs and data from attack, damage or unauthorized access.
Passwords A secret word or phrase that one uses to access private or personal information.
Encryption The process of encoding a message so that only intended recipients can access it.
Decryption The process of reversing encryption.
Hacker An unauthorized user who gains access to and sometimes tampers with information in a computer system
Misinformation False information that is portrayed as being true.
Server A computer that is designed to serve information to other computers.
Malware Software intentionally designed to cause damage to a computer, server or computer network.
Spoofing Vis ac commodo adipiscing arcu aliquet.
Social Engineering A situation in which a person or program successfully masquerades as another by falsifying data, to gain an illegitimate advantage.
Phishing An attempt to gain passwords and other sensitive information by disguising as someone the target knows.
Bot A software application that runs automated tasks.
Vulnerability The state of being weak or poorly defended. Used to describe weak networks.
Bug A problem in software that causes the program to yield unexpected results.
Patch A software update that fixes a bug.
Cookie A small file stored in browsers that holds personally identifiable information to remember users' preferences.
Keylogger Malware that monitors users keystrokes and sends them to unauthorized attackers.
Screenshot A still photo of a computer screen.

For Parents

Carnegie Cyber Academy is a program developed by Carnegie Mellon University's Information Networking Institute. As the internet becomes increasingly ubiquitous, it's becoming more and more important to educate minors on how they can stay safe online. The Academy was built in 2010, and has since become one of the premier sources of online cyber security information for students looking to get started on learning basic cyber practices.
